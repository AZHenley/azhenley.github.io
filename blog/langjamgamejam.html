Langjam Gamejam: Build a programming language and then use it to make a game in 7 days

11/28/2025

My favorite time to code useless stuff is in December. There's just nothing like avoiding cold weather during the holiday break while coding.

I figured out what I'm building this year: I'm going to make a compiler for my own programming language and then use that language to make a game. And I'm going to do it in 7 days.

I told a few friends about it and they got excited to join in the fun, so I'm opening it up to anyone. I'm calling it Langjam Gamejam.

I made a website with the details, https://langjamgamejam.com, and set up a jam on Itch.io for the submissions.

There really aren't any rules. You decide what a language is and what a game is, and you use whatever technologies you want. Make a language and then make a game with it. December 14–20. Do something creative.

Let me know if you join!
