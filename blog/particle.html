The Init event runs once. The Update event runs each frame for each particle. Each particle will be moved by its speed in the direction of its rotation. You can modify all individual particle's properties through the p object, like p.speed. You can even store your own variables, like p.foobar. The other properties of particles include:

  • x,y -- position
  • xscale,yscale -- multiplies the particle's size
  • r,g,b -- color values from 0 to 255
  • a -- alpha from 0 to 1
  • life -- how many frames until the particle is deleted
  • speed -- how far to be moved at the end of the frame
  • rotation -- the angle to be moved at the end of the frame, in radians
  • i -- the unique id of the particle, starting at 0 and counting up
  • z -- the depth to draw the object, smaller numbers get drawn first

Press Start to restart with the current code. You can modify Burst Size and Burst Delay anytime with the sliders. There are these global variables (I recommend using them as read-only):

  • time -- number of frames passed
  • alive -- number of particles currently alive
  • burstsize -- number of particles to create at a time
  • burstdelay -- delay between bursts
There is also a global time variable that increments each frame.
